Os Principais Comandos para Criar um Site

1. Estrutura Básica do HTML

O comando <html> é o elemento raiz de um documento HTML. Dentro dele, o <head> contém metadados e o <body> contém o conteúdo visível da página. Veja o exemplo:

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Título da Página</title>
</head>
<body>
</body>
</html>

2. Comando CSS

O comando color no CSS é utilizado para definir a cor do texto. Além disso, o font-family define a fonte utilizada. Veja o exemplo:

p {
color: black;
font-family: 'Times New Roman', serif;
}
